### Task 2: Lark LALR Grammar (`sas.lark`)

**File**: `partition/chunking/sas.lark`

This is the **full** boundary detection grammar (not the minimal pre-validator from Week 1–2). It must correctly identify all 9 partition types.

```lark
// SAS Block Boundary Grammar
// Purpose: Detect canonical SAS block boundaries from source code
// Coverage: ~80% of blocks resolved deterministically

start: (block | _IGNORABLE)*

block: data_step
     | proc_sql_block
     | proc_block
     | macro_def
     | macro_invocation
     | conditional_block
     | loop_block
     | global_statement
     | include_reference

// DATA step — ends with RUN;
data_step: DATA_KW NAME_LIST ";" body RUN_END
DATA_KW: /(?i)\bDATA\b/
RUN_END: /(?i)\bRUN\s*;/

// PROC SQL — ends with QUIT;
proc_sql_block: PROC_SQL_KW body QUIT_END
PROC_SQL_KW: /(?i)\bPROC\s+SQL\b\s*;/
QUIT_END: /(?i)\bQUIT\s*;/

// PROC other — ends with RUN; or QUIT;
proc_block: PROC_KW PROC_NAME body (RUN_END | QUIT_END)
PROC_KW: /(?i)\bPROC\b/
PROC_NAME: /\s+\w+\s*;/

// %MACRO ... %MEND
macro_def: MACRO_START body MEND_END
MACRO_START: /(?i)%MACRO\s+\w+[^;]*;/
MEND_END: /(?i)%MEND\s*\w*\s*;/

// Macro invocation: %name(...)
macro_invocation: MACRO_CALL_PATTERN
MACRO_CALL_PATTERN: /(?i)%\w+\s*\([^)]*\)\s*;?/

// %IF/%THEN/%ELSE blocks
conditional_block: IF_START body (ELSE_CLAUSE body)? END_STMT
IF_START: /(?i)%IF\b/
ELSE_CLAUSE: /(?i)%ELSE\b/
END_STMT: /(?i)%END\s*;/

// %DO loops
loop_block: DO_START body END_STMT
DO_START: /(?i)%DO\b[^;]*;/

// Global statements (single line)
global_statement: GLOBAL_KW /[^;]*/ ";"
GLOBAL_KW: /(?i)\b(OPTIONS|LIBNAME|FILENAME|TITLE\d?|FOOTNOTE\d?)\b/

// %INCLUDE
include_reference: INCLUDE_KW /[^;]*/ ";"
INCLUDE_KW: /(?i)%INCLUDE\b/

// Helpers
NAME_LIST: /[^;]+/
body: /[\s\S]*?/  // non-greedy match
_IGNORABLE: /\s+/ | COMMENT
COMMENT: /\/\*[\s\S]*?\*\// | /\*[^;]*;/
```

> **Important**: Lark's LALR parser may struggle with the full SAS grammar. In practice, build a **custom boundary detector** that uses regex + state from StateAgent, with Lark as a validation step. The grammar above is a reference — your actual implementation will likely be a hybrid.

**Practical implementation** (recommended approach):

```python
import re
from partition.chunking.models import BlockBoundaryEvent
from partition.streaming.models import LineChunk, ParsingState
from partition.models.enums import PartitionType

class BoundaryDetector:
    """
    Deterministic boundary detector using StateAgent output.
    No LLM — pure pattern matching on ParsingState transitions.
    """

    # Mapping from StateAgent block types to PartitionType
    TYPE_MAP = {
        "DATA_STEP": PartitionType.DATA_STEP,
        "PROC_BLOCK": PartitionType.PROC_BLOCK,
        "SQL_BLOCK": PartitionType.SQL_BLOCK,
        "MACRO_DEFINITION": PartitionType.MACRO_DEFINITION,
        "MACRO_INVOCATION": PartitionType.MACRO_INVOCATION,
        "CONDITIONAL_BLOCK": PartitionType.CONDITIONAL_BLOCK,
        "LOOP_BLOCK": PartitionType.LOOP_BLOCK,
        "GLOBAL_STATEMENT": PartitionType.GLOBAL_STATEMENT,
        "INCLUDE_REFERENCE": PartitionType.INCLUDE_REFERENCE,
    }

    COVERAGE_MAP = {
        PartitionType.DATA_STEP: "full",
        PartitionType.PROC_BLOCK: "full",
        PartitionType.MACRO_DEFINITION: "full",
        PartitionType.MACRO_INVOCATION: "full",
        PartitionType.SQL_BLOCK: "full",
        PartitionType.CONDITIONAL_BLOCK: "structural_only",
        PartitionType.LOOP_BLOCK: "structural_only",
        PartitionType.GLOBAL_STATEMENT: "structural_only",
        PartitionType.INCLUDE_REFERENCE: "structural_only",
    }

    def detect(self, chunks_with_states, file_id, trace_id):
        """
        Process a list of (LineChunk, ParsingState) tuples.
        Emit BlockBoundaryEvent whenever a block closes.
        """
        events = []
        current_block_lines = []
        current_type = None
        block_start = None

        for chunk, state in chunks_with_states:
            if state.current_block_type and state.current_block_type != "IDLE":
                if current_type is None:
                    # New block started
                    current_type = state.current_block_type
                    block_start = chunk.line_number
                current_block_lines.append(chunk.content)
            elif current_type is not None:
                # Block just ended
                pt = self.TYPE_MAP.get(current_type)
                if pt:
                    raw_code = '\n'.join(current_block_lines)

                    # Ambiguity check: > 200 lines with no clear end
                    is_ambiguous = len(current_block_lines) > 200

                    event = BlockBoundaryEvent(
                        file_id=file_id,
                        partition_type=pt,
                        line_start=block_start,
                        line_end=chunk.line_number,
                        raw_code=raw_code,
                        boundary_method="lark",
                        is_ambiguous=is_ambiguous,
                        nesting_depth=state.nesting_depth,
                        macro_scope=dict(state.variable_scope),
                        variable_scope=dict(state.variable_scope),
                        dependency_refs=list(state.active_dependencies),
                        test_coverage_type=self.COVERAGE_MAP.get(pt, "full"),
                        trace_id=trace_id,
                    )
                    events.append(event)

                current_block_lines = []
                current_type = None
                block_start = None

        return events
```

### Task 3: BoundaryDetectorAgent (#6)

**File**: `partition/chunking/boundary_detector_agent.py`

**Inputs**: `list[(LineChunk, ParsingState)]` from the streaming pipeline  
**Outputs**: `list[BlockBoundaryEvent]`

This agent orchestrates:
1. **Deterministic pass** (BoundaryDetector) → handles ~80% of blocks
2. **LLM resolver pass** → handles ~20% ambiguous blocks
3. **Combines and returns** all events sorted by `line_start`

```python
class BoundaryDetectorAgent(BaseAgent):
    agent_name = "BoundaryDetectorAgent"

    def __init__(self, trace_id=None):
        super().__init__(trace_id)
        self.detector = BoundaryDetector()
        self.llm_resolver = LLMBoundaryResolver(trace_id=self.trace_id)

    async def process(self, chunks_with_states, file_id):
        # Step 1: Deterministic detection
        events = self.detector.detect(
            chunks_with_states, file_id, self.trace_id
        )

        # Step 2: Route ambiguous blocks to LLM
        resolved = []
        for event in events:
            if event.is_ambiguous:
                self.logger.info("routing_to_llm",
                    line_start=event.line_start,
                    line_end=event.line_end)
                try:
                    llm_event = await self.llm_resolver.resolve(event)
                    resolved.append(llm_event)
                except Exception:
                    # Fallback: keep lark result
                    event.boundary_method = "lark_fallback"
                    resolved.append(event)
            else:
                resolved.append(event)

        # Step 3: Sort by line_start
        resolved.sort(key=lambda e: e.line_start)

        self.logger.info("boundary_detection_complete",
            total=len(resolved),
            lark=sum(1 for e in resolved if e.boundary_method == "lark"),
            llm=sum(1 for e in resolved if e.boundary_method == "llm_8b"),
            fallback=sum(1 for e in resolved if e.boundary_method == "lark_fallback"))

        return resolved
```

```

**Implementation**:

```python
import instructor
from ollama import AsyncClient
import tiktoken

class LLMBoundaryResolver:
    MAX_TOKENS = 6000
    MODEL = "llama3.1:8b"

    def __init__(self, trace_id=None):
        self.trace_id = trace_id
        self.client = instructor.from_openai(
            AsyncClient(host="http://localhost:11434"),
            mode=instructor.Mode.JSON
        )
        self.tokenizer = tiktoken.get_encoding("cl100k_base")

    async def resolve(self, event: BlockBoundaryEvent) -> BlockBoundaryEvent:
        """Use LLM to determine the correct block type for an ambiguous block."""

        # Tiktoken guard: truncate if too long
        raw_code = event.raw_code
        tokens = self.tokenizer.encode(raw_code)
        if len(tokens) > self.MAX_TOKENS - 500:
            raw_code = self.tokenizer.decode(tokens[:self.MAX_TOKENS - 500])
            raw_code += "\n... [TRUNCATED]"

        prompt = f"""Analyze this SAS code block and determine its type.

SAS Code:
```sas
{raw_code}
```

Determine the block type from these options:
- DATA_STEP: starts with DATA statement, ends with RUN;
- PROC_BLOCK: starts with PROC (not SQL), ends with RUN; or QUIT;
- SQL_BLOCK: PROC SQL, ends with QUIT;
- MACRO_DEFINITION: %MACRO ... %MEND
- MACRO_INVOCATION: %macroname() call
- CONDITIONAL_BLOCK: %IF/%THEN/%ELSE outside PROC
- LOOP_BLOCK: %DO/%END iterative
- GLOBAL_STATEMENT: OPTIONS, LIBNAME, FILENAME, TITLE
- INCLUDE_REFERENCE: %INCLUDE

Also determine the exact line where this block ends.
Respond with JSON: {{"block_type": "...", "line_end": N, "confidence": 0.0-1.0}}"""

        # Use instructor for typed output
        response = await self.client.chat.completions.create(
            model=self.MODEL,
            messages=[{"role": "user", "content": prompt}],
            response_model=LLMBoundaryResponse,
            max_retries=3,
        )

        # Update the event
        event.partition_type = PartitionType(response.block_type)
        event.line_end = response.line_end
        event.confidence = response.confidence
        event.boundary_method = "llm_8b"
        event.is_ambiguous = False

        return event


class LLMBoundaryResponse(BaseModel):
    block_type: str
    line_end: int
    confidence: float
```

**Retry policy**: 3× exponential backoff (base 1s). On total failure → keep the Lark result with `boundary_method="lark_fallback"`.

class PartitionBuilderAgent(BaseAgent):

                source_file_id=event.file_id,
                content_hash=content_hash,

### Task 6: Benchmark Evaluation Script

**File**: `benchmark/boundary_benchmark.py`

```python
"""
Evaluate boundary detection accuracy against the gold standard corpus.
"""
import json
from pathlib import Path

def evaluate_boundary_accuracy(detected_events, gold_dir):
    """
    Compare detected BlockBoundaryEvents against gold standard annotations.

    A boundary is "correct" if:
    - partition_type matches EXACTLY
    - line_start is within ±2 lines
    - line_end is within ±2 lines
    """
    total_gold = 0
    correct = 0
    type_confusion = {}  # {(predicted, actual): count}

    for gold_file in Path(gold_dir).glob("*.gold.json"):
        with open(gold_file) as f:
            gold = json.load(f)

        file_events = [e for e in detected_events
                       if str(e.file_id) == gold.get("file_id")]

        for gold_block in gold["blocks"]:
            total_gold += 1
            matched = False
            for event in file_events:
                if (event.partition_type.value == gold_block["partition_type"]
                    and abs(event.line_start - gold_block["line_start"]) <= 2
                    and abs(event.line_end - gold_block["line_end"]) <= 2):
                    correct += 1
                    matched = True
                    break

            if not matched:
                # Find closest event for confusion matrix
                closest = min(file_events,
                    key=lambda e: abs(e.line_start - gold_block["line_start"]),
                    default=None)
                if closest:
                    key = (closest.partition_type.value,
                           gold_block["partition_type"])
                    type_confusion[key] = type_confusion.get(key, 0) + 1

    accuracy = correct / total_gold if total_gold > 0 else 0
    return {
        "accuracy": accuracy,
        "correct": correct,
        "total": total_gold,
        "target": 0.90,
        "passed": accuracy >= 0.90,
        "confusion": type_confusion,
    }
```
