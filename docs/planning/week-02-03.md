## Tasks

### Task 1: StreamAgent (#4)

**File**: `partition/streaming/stream_agent.py`

**Inputs**: `FileMetadata` (from FileRegistry — provides `file_path` and `encoding`)  
**Outputs**: `AsyncIterator[LineChunk]`

**LineChunk data model**:
```python
from pydantic import BaseModel
from uuid import UUID

class LineChunk(BaseModel):
    file_id: UUID
    line_number: int
    content: str           # the raw line text (stripped of trailing newline)
    byte_offset: int       # cumulative byte position in the file
    is_continuation: bool  # True if line ends with no semicolon (SAS continuation)
```

**Step-by-step logic**:

1. **Open file** with `aiofiles.open(file_path, mode='r', encoding=encoding)`.
2. **Read line-by-line** using `async for line in f:`.
3. **Internal buffer**: Accumulate lines until a logical SAS statement boundary (line containing `;`). This coalesces multi-line SAS statements into single `LineChunk` objects when needed.
4. **Yield** `LineChunk` objects one by one.
5. **Backpressure**: Push each chunk to `asyncio.Queue(maxsize=200)` — if the queue is full, the producer blocks (`await queue.put(chunk)`).
6. **Large file adjustment**:
   - File > 100 MB: `maxsize = 50`
   - File > 500 MB: Switch to sequential processing (5,000-line segments)
7. **EOF**: Yield any remaining buffered content, then signal completion.

**Key async pattern**:
```python
import aiofiles
import asyncio
from partition.base_agent import BaseAgent

class StreamAgent(BaseAgent):
    agent_name = "StreamAgent"

    def __init__(self, queue: asyncio.Queue, trace_id=None):
        super().__init__(trace_id)
        self.queue = queue

    async def process(self, file_meta: FileMetadata):
        self.logger.info("streaming_start", file=file_meta.file_path)
        line_num = 0
        byte_offset = 0
        buffer = ""

        async with aiofiles.open(
            file_meta.file_path,
            mode='r',
            encoding=file_meta.encoding
        ) as f:
            async for raw_line in f:
                line_num += 1
                byte_offset += len(raw_line.encode(file_meta.encoding))
                buffer += raw_line

                # SAS statement boundary: line contains semicolon
                if ';' in raw_line:
                    chunk = LineChunk(
                        file_id=file_meta.file_id,
                        line_number=line_num,
                        content=buffer.strip(),
                        byte_offset=byte_offset,
                        is_continuation=False,
                    )
                    await self.queue.put(chunk)  # blocks if queue full
                    buffer = ""

        # Flush remaining buffer
        if buffer.strip():
            chunk = LineChunk(
                file_id=file_meta.file_id,
                line_number=line_num,
                content=buffer.strip(),
                byte_offset=byte_offset,
                is_continuation=True,
            )
            await self.queue.put(chunk)

        # Sentinel to signal EOF
        await self.queue.put(None)
        self.logger.info("streaming_complete", lines=line_num)
```

### Task 2: StateAgent (#5)

**File**: `partition/streaming/state_agent.py`

**Inputs**: `LineChunk` (consumed from the queue)  
**Outputs**: `ParsingState` (updated per line)

**ParsingState data model**:
```python
from pydantic import BaseModel
from typing import Optional
from partition.models.enums import PartitionType

class ParsingState(BaseModel):
    current_block_type: Optional[str] = None    # "IDLE" | PartitionType values
    block_start_line: Optional[int] = None
    nesting_depth: int = 0
    macro_stack: list[str] = []                  # push on %MACRO, pop on %MEND
    variable_scope: dict[str, str] = {}          # var_name → last_assigned_type
    active_dependencies: list[str] = []          # LIBNAME refs, %INCLUDE targets
    in_comment: bool = False                     # block comment tracking
    in_string: bool = False                      # multi-line string tracking
```

**Step-by-step FSM logic**:

The StateAgent is a **pure Python FSM** — no LLM, no I/O, no network. It must process each line in **< 0.5 ms**.

```python
import re

class StateAgent(BaseAgent):
    agent_name = "StateAgent"

    # Compiled regex patterns for speed
    DATA_START = re.compile(r'^\s*DATA\s+', re.IGNORECASE)
    PROC_START = re.compile(r'^\s*PROC\s+', re.IGNORECASE)
    PROC_SQL   = re.compile(r'^\s*PROC\s+SQL\b', re.IGNORECASE)
    MACRO_DEF  = re.compile(r'^\s*%MACRO\s+(\w+)', re.IGNORECASE)
    MACRO_END  = re.compile(r'^\s*%MEND\b', re.IGNORECASE)
    MACRO_CALL = re.compile(r'%(\w+)\s*\(', re.IGNORECASE)
    DO_START   = re.compile(r'^\s*%DO\b', re.IGNORECASE)
    IF_START   = re.compile(r'^\s*%IF\b', re.IGNORECASE)
    END_STMT   = re.compile(r'^\s*%END\b', re.IGNORECASE)
    RUN_STMT   = re.compile(r'^\s*RUN\s*;', re.IGNORECASE)
    QUIT_STMT  = re.compile(r'^\s*QUIT\s*;', re.IGNORECASE)
    INCLUDE    = re.compile(r'%INCLUDE\s+', re.IGNORECASE)
    GLOBAL     = re.compile(r'^\s*(OPTIONS|LIBNAME|FILENAME|TITLE)\b', re.IGNORECASE)
    COMMENT_S  = re.compile(r'/\*')
    COMMENT_E  = re.compile(r'\*/')
    RETAIN     = re.compile(r'\bRETAIN\b', re.IGNORECASE)
    MERGE      = re.compile(r'\bMERGE\b', re.IGNORECASE)
    BY_STMT    = re.compile(r'\bBY\b', re.IGNORECASE)
    FIRST_LAST = re.compile(r'\b(FIRST|LAST)\.\w+', re.IGNORECASE)

    def __init__(self, trace_id=None):
        super().__init__(trace_id)
        self.state = ParsingState()

    async def process(self, chunk: LineChunk) -> ParsingState:
        line = chunk.content

        # 1. Handle block comments
        if self.state.in_comment:
            if self.COMMENT_E.search(line):
                self.state.in_comment = False
            return self.state
        if self.COMMENT_S.search(line) and not self.COMMENT_E.search(line):
            self.state.in_comment = True
            return self.state

        # 2. Block transitions
        if self.state.current_block_type is None or self.state.current_block_type == "IDLE":
            self._detect_block_start(line, chunk.line_number)
        else:
            self._check_block_end(line)

        # 3. Track nesting
        if self.DO_START.search(line) or self.IF_START.search(line):
            self.state.nesting_depth += 1
        if self.END_STMT.search(line):
            self.state.nesting_depth = max(0, self.state.nesting_depth - 1)

        # 4. Track macro stack
        m = self.MACRO_DEF.search(line)
        if m:
            self.state.macro_stack.append(m.group(1))
        if self.MACRO_END.search(line) and self.state.macro_stack:
            self.state.macro_stack.pop()

        # 5. Track dependencies
        if self.INCLUDE.search(line):
            self.state.active_dependencies.append(line.strip())

        # 6. Track variable assignments (simplified)
        # Look for assignment patterns: var = expr ;
        assign_match = re.search(r'\b(\w+)\s*=\s*', line)
        if assign_match and self.state.current_block_type:
            self.state.variable_scope[assign_match.group(1)] = "assigned"

        return self.state

    def _detect_block_start(self, line: str, line_num: int):
        if self.DATA_START.match(line):
            self.state.current_block_type = "DATA_STEP"
            self.state.block_start_line = line_num
        elif self.PROC_SQL.match(line):
            self.state.current_block_type = "SQL_BLOCK"
            self.state.block_start_line = line_num
        elif self.PROC_START.match(line):
            self.state.current_block_type = "PROC_BLOCK"
            self.state.block_start_line = line_num
        elif self.MACRO_DEF.match(line):
            self.state.current_block_type = "MACRO_DEFINITION"
            self.state.block_start_line = line_num
        elif self.MACRO_CALL.search(line):
            self.state.current_block_type = "MACRO_INVOCATION"
            self.state.block_start_line = line_num
        elif self.GLOBAL.match(line):
            self.state.current_block_type = "GLOBAL_STATEMENT"
            self.state.block_start_line = line_num
        elif self.INCLUDE.match(line):
            self.state.current_block_type = "INCLUDE_REFERENCE"
            self.state.block_start_line = line_num

    def _check_block_end(self, line: str):
        bt = self.state.current_block_type
        if bt in ("DATA_STEP", "PROC_BLOCK") and self.RUN_STMT.search(line):
            self._emit_block_end()
        elif bt == "SQL_BLOCK" and self.QUIT_STMT.search(line):
            self._emit_block_end()
        elif bt == "MACRO_DEFINITION" and self.MACRO_END.search(line):
            self._emit_block_end()
        elif bt in ("GLOBAL_STATEMENT", "INCLUDE_REFERENCE"):
            if ';' in line:
                self._emit_block_end()

    def _emit_block_end(self):
        self.state.current_block_type = "IDLE"
        self.state.block_start_line = None
```

### Task 3: BackpressureQueue

**File**: `partition/streaming/backpressure.py`

```python
import asyncio
from partition.models.file_metadata import FileMetadata

def create_queue(file_meta: FileMetadata) -> asyncio.Queue:
    """Create a backpressure queue with size based on file size."""
    size_bytes = file_meta.file_size_bytes
    if size_bytes > 500_000_000:  # 500 MB
        maxsize = 10   # sequential processing mode
    elif size_bytes > 100_000_000:  # 100 MB
        maxsize = 50
    else:
        maxsize = 200  # standard
    return asyncio.Queue(maxsize=maxsize)
```

### Task 4: Integration — Producer/Consumer Pipeline

**File**: `partition/streaming/pipeline.py`

Wire StreamAgent (producer) and StateAgent (consumer) together:

```python
import asyncio
from partition.streaming.stream_agent import StreamAgent
from partition.streaming.state_agent import StateAgent
from partition.streaming.backpressure import create_queue

async def run_streaming_pipeline(file_meta):
    queue = create_queue(file_meta)
    stream = StreamAgent(queue=queue)
    state = StateAgent()

    results = []

    async def producer():
        await stream.process(file_meta)

    async def consumer():
        while True:
            chunk = await queue.get()
            if chunk is None:  # EOF sentinel
                break
            parsing_state = await state.process(chunk)
            results.append((chunk, parsing_state.model_copy()))
            queue.task_done()

    await asyncio.gather(producer(), consumer())
    return results
```

## Common Pitfalls

1. **`async for` on aiofiles**: Make sure aiofiles version supports `async for line in f`. If not, use `await f.readline()` in a loop.
2. **SAS continuation lines**: SAS allows multi-line statements. A line without `;` is a continuation. Buffer them before yielding.
3. **Block comment tracking**: `/* ... */` can span multiple lines. Track `in_comment` state to avoid false block detections inside comments.
4. **PROC SQL special case**: `PROC SQL` ends with `QUIT;` not `RUN;`. Handle this explicitly.
5. **Macro nesting edge case**: `%MEND` without a preceding `%MACRO` should not crash — just `max(0, depth - 1)`.
6. **Performance**: Do NOT use regex for every line if a simple string check suffices. E.g., `if 'DATA ' in line.upper()` is faster than `re.match(r'DATA\s+', line, re.I)`.
